# Security and disclosure policy

This repository is a deliberately sanitized public portfolio. It does not contain the private SoherDocs or Pi-extension source trees.

## Reporting accidental disclosure

If you believe this repository exposes a credential, private path, personal record, internal prompt, or confidential implementation detail, contact the repository owner privately through the LinkedIn profile in the README. Do not open a public issue containing the suspected material.

## Supervised private review

Private source may be shown during a supervised technical session. Temporary read-only access can be arranged for an identified technical interviewer after agreeing:

- which repository and modules are in scope;
- the review period and revocation date;
- that credentials, private prompts, personal notes, and unrelated projects remain excluded;
- that source will not be copied, redistributed, or used outside the interview process.

Temporary access is not guaranteed and may be replaced with a live walkthrough when repository-level access would expose unrelated private material.
